Web semantica

Elementos con significado!

Ejemplos de etiquetas no semánticas: Div, Span.
Ejemplos de etiquetas semánticas: Table, Form, Article

Header

Es el encabezado de una página, sección o artículo. incluye elementos como el logo, texto introductorio, menús de navegación. Puede haber múltiples headers en una página, pero no puede haber un header dentro de un footer o de otro header.

Footer

Es el pié de página de un sitio o sección. Ouede ser el footer de la web o de un artículo puede haber múltiples por página. puede incluir información como:
  • Información de contacto
  • Enlaces a las RRSS
  • Mapa del sitio
  • Créditos
  • Formulario de contacto

Navegación

Define un conjunto de enlaces de navegación. no todos los links deben estar dentro de un "Nav".

Main

Crea un contenedor para el contenido principal de nuestra página. Este debe ser único y no debe incluir contenido repetido de otras secciones, como por ejemplo del sidebar, logos, etc. No puede ser hijo de etiquetas como: areticle, aside, footer, header o nav.

Aside

Define el contenido relacionado indirectamente con el contenido principal, como por ejemplo un sidebar. Elementos que solemos poner en un aside:
  • Publicidad
  • Biografía del autor
  • Enlaces relacionados
  • Información de perfil

Section

Define una sección en una página donde se agrupan los contenidos relacionados, y normalmente incluyen un título. Una web se compone de varias secciones, cada una con su contenido y propósito:
Ejemplos de uso:
  • Capítulos
  • Introducción
  • Noticias
  • Información de contacto

Article

Define un contenido independiente. El mismo puede ser compartido o reutilizado en otro sítio sin requerir contexto
Tambien suele incluir un título.
  • Un post de un blog
  • Un artículo d euna revista
  • Comentario de usuario (Reseña)
  • Tarjeta de productos

Puedo encontrar artículos dentro de seccione sy secciones dentro de artículos. Un artículo puede tener multiples secciones. Una sección puede tener múltiples artículos.

Figure

defien elementos autocontenidos, como puede ser una imagen, un diagrama, un gráfico, una tabla o una foto
Dentro de ella se suele utilizar la etiqueta "figcaption" para describir el contenido, esto es además del ya incluido "alt" en caso de imágenes. Ej: Wikipedia.

Mark

Define un texto resaltado. Normalmente se muestra con Fondo amarillo.
