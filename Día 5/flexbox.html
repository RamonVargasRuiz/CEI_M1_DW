Justify content

El justify-content es una propiedad de CSS que se usa dentro de un contenedor flexbox o grid para alinear los elementos en el eje principal (horizontal en flex-direction: row y vertical en flex-direction: column).

order

La propiedad order se usa en Flexbox para cambiar el orden visual de los elementos dentro del contenedor, sin alterar el HTML. Por defecto, todos los elementos tienen order: 0. Los elementos con un número menor aparecen antes y los de número mayor aparecen después.
